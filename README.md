
# 🛒 E-Commerce Website

A modern and responsive **E-Commerce Web Application** built using modern web technologies. This project provides a seamless online shopping experience with product browsing, cart management, and checkout functionality.

---

## 🚀 Features

* 🛍️ Browse products with detailed descriptions
* 🔍 Search and filter products
* 🛒 Add/remove items from cart
* 💳 Checkout functionality
* 👤 User authentication (Login/Register)
* 📦 Order management
* 📱 Fully responsive design

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/rge5/e-commerce.git
cd e-commerce
```

2. Install dependencies:

```bash
npm install
```

3. Run the project:

```bash
npm start
```

4. Open in browser:

```
http://localhost:3000
```
